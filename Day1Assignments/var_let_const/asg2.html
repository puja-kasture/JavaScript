Difference between var, let and const

var:

1. Containers for storing values with unique names.
2. Like as global variable.
3. Function scope variable


let:

1. Provide Block Scope variables
2. Local variable
3. Means the scope will be remain upto that particular block of code only


const:

1. Define variable as constant
2. Means value given to particular variable can't change
